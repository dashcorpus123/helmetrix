Project Title: Helmetrix: IoT-Enabled Smart Helmet Storage Solution
Description:Helmetrix is an innovative IoT-powered mobile application that provides secure, convenient storage solutions for motorcycle helmets. The system addresses the common challenge faced by ridersâ€”finding a safe place to store helmets when visiting campuses, workplaces, or public venues. By combining smart locker technology with a user-friendly mobile interface, Helmetrix ensures helmets are stored safely, hygienically, and are easily accessible through a contactless QR code system. The application features real-time locker monitoring, automated access control, and comprehensive activity logging to deliver a seamless user experience.
Technologies Used: Flutter, Dart, IoT, Cloud Computing (android studio)
Features:Real-time Locker Management System - Visual dashboard displaying live status of all lockers (available, occupied, reserved)
User Authentication - Secure login and registration system with Firebase Authentication
QR Code Access System - Contactless locker access and helmet retrieval through secure QR scanning
Quick Actions Dashboard - Streamlined locker selection and rental controls for efficient booking
IoT Device Monitoring - Real-time tracking of solenoid locks, magnetic sensors, and LED indicators
Helmet Compatibility Guide - Information system showing supported helmet types and sizes for each locker
Activity Logging - Comprehensive tracking of locker usage, access times, rental history, and transactions
User Guide Section - Interactive step-by-step instructions for first-time users
System Status Display - Application version information and IoT hardware connection status
Contact & Support - In-app support information and physical location details
Push Notifications - Real-time alerts for rental confirmations, time reminders, and system updates
Navigation Menu - Intuitive tab-based navigation for seamless user experience
Installation Instructions: Flutter SDK (version 3.0 or higher)
Dart SDK (version 2.17 or higher)
Android Studio or VS Code with Flutter extensions
Setup: ESP32 microcontroller (WiFi-enabled)
Solenoid locks (one per locker compartment)
